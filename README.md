# 答题小程序云开发版
使用小程序原生云开发搭建，
界面使用ColorUI


+目标功能：
    +由原生云数据库储存题库
    +随机出题
    +回答评分
    +练习题目
    +预约考试
    

+目前以实现功能
    随机从题库中抽取题目，以及答题页面的实现
